# 🍽️ Savoria -- Restaurant Landing Page

A modern and responsive **restaurant landing page** built using **HTML,
CSS, and JavaScript**.\
This project demonstrates a clean UI design for restaurants including
menu display, popular dishes, gallery, and a reservation system.

------------------------------------------------------------------------

# 🚀 Features

-   Responsive Navigation Bar
-   Hero Section with Call‑to‑Action buttons
-   Popular Dishes Section
-   About Restaurant Section
-   Interactive Menu Filtering
-   Image Gallery
-   Table Reservation Form
-   Modern Footer with Contact Information
-   Fully Responsive Design

------------------------------------------------------------------------

# 🖥️ Technologies Used

-   HTML5
-   CSS3
-   JavaScript (Vanilla JS)

------------------------------------------------------------------------

# 📸 Sections Included

## 🏠 Home

Hero section with a background image and action buttons.

## 🍕 Popular Dishes

Dynamic dish cards loaded using JavaScript Fetch API.

## 📖 About

Restaurant story, statistics, and values section.

## 🍔 Menu

Interactive menu filtering by category:

-   Starters
-   Mains
-   Desserts
-   Drinks

## 🖼️ Gallery

Restaurant interior and food showcase.

## 📅 Reservation

Table booking form where users can enter:

-   Name
-   Email
-   Phone Number
-   Number of Guests
-   Date
-   Time
-   Special Requests

------------------------------------------------------------------------
